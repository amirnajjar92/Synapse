# PDF Generation Prompts for Synapse Knowledge Base

## 🇺🇸 ENGLISH VERSION PROMPT

```
I need you to convert the attached Markdown knowledge base document into a professional, 
investor-ready PDF with the following specifications:

**FORMATTING REQUIREMENTS:**

1. **Cover Page:**
   - Title: "Synapse - Complete Platform Knowledge Base"
   - Subtitle: "AI-Powered Fitness & Health Operating System"
   - Tagline: "Your AI Fitness Brain"
   - Logo placeholder with description: "Orange and blue neural network design"
   - Document version and date
   - Professional gradient background (dark to lighter)

2. **Table of Contents:**
   - Clickable links to all major sections
   - Page numbers
   - 2-level hierarchy (main sections and subsections)

3. **Typography:**
   - Headings: Sans-serif, bold, hierarchical sizes (H1: 24pt, H2: 18pt, H3: 14pt)
   - Body text: 11pt, professional font (Open Sans, Helvetica, or similar)
   - Code blocks: Monospace font with light gray background
   - Line spacing: 1.5 for body text
   - Margins: 1 inch all sides

4. **Color Scheme:**
   - Primary color: #FC4C02 (Orange) for headings and accents
   - Secondary color: #3B82F6 (Blue) for links and secondary accents
   - Background: White with subtle gray (#F9FAFB) for tables/code blocks
   - Text: Dark gray (#1F2937) for body, Black (#000000) for headings

5. **Visual Elements:**
   - Add icons next to major section headings (emoji or simple graphics)
   - Tables: Alternating row colors, borders, header row with background
   - Lists: Bullet points with orange markers for emphasis
   - Callout boxes: Light orange background for key insights
   - Page headers: Section name on left, page number on right
   - Page footers: "Synapse Knowledge Base" on left, date on right

6. **Special Formatting:**
   - Highlight competitive advantages with checkmark icons (✓)
   - Use charts/graphs for metrics if possible (or describe where they'd go)
   - Add subtle horizontal lines between major sections
   - Use blockquotes with left border for key quotes
   - Feature boxes for user personas with avatar placeholders

7. **Professional Touches:**
   - Executive summary on page 2 (1-page overview)
   - Section divider pages with large section titles
   - Consistent spacing and alignment throughout
   - No orphaned headings (heading at bottom of page)
   - Hyphenation for justified text

8. **Content Enhancements:**
   - Add relevant icons/emojis next to feature lists
   - Create visual hierarchy with boxes and borders
   - Highlight important metrics in larger text or callouts
   - Use comparison tables for competitive analysis
   - Add "Key Takeaway" boxes at end of major sections

**OUTPUT REQUIREMENTS:**
- Format: PDF (A4 size)
- Pages: Approximately 40-50 pages with proper formatting
- File size: Optimized for sharing (under 10MB)
- Accessibility: Tagged PDF with proper heading structure
- Bookmarks: Enable PDF bookmarks for navigation

Please create a polished, professional document suitable for:
- Investor presentations
- Partner negotiations
- Co-founder onboarding
- Stakeholder meetings
- Strategic planning sessions

Make it visually engaging while maintaining corporate professionalism.
```

   - پاصفحه: "پایگاه دانش سیناپس" در راست، تاریخ در چپ

6. **قالب‌بندی ویژه:**
   - برجسته کردن مزایای رقابتی با آیکون تیک (✓)
   - استفاده از نمودارها برای معیارها در صورت امکان
   - افزودن خطوط افقی ملایم بین بخش‌های اصلی
   - استفاده از نقل قول‌ها با حاشیه چپ برای جملات کلیدی

7. **لمس‌های حرفه‌ای:**

   - بدون سرفصل‌های یتیم (سرفصل در پایین صفحه)

8. **بهبودهای محتوا (به فارسی):**
   - ترجمه کامل تمام محتوا به فارسی روان و طبیعی
   - استفاده از اصطلاحات فنی فارسی در کنار معادل انگلیسی در پرانتز
   - افزودن آیکون‌های مرتبط در کنار لیست ویژگی‌ها
   - ایجاد سلسله مراتب بصری با جعبه‌ها و حاشیه‌ها
   - برجسته کردن معیارهای مهم با متن بزرگ‌تر
   - استفاده از جداول مقایسه برای تحلیل رقابتی
   - افزودن جعبه‌های "نکته کلیدی" در پایان بخش‌های اصلی

**نکات مهم برای ترجمه:**
- ترجمه تخصصی و فنی، نه ترجمه ماشینی
- حفظ معنی و مفهوم اصلی
- استفاده از اصطلاحات رایج در صنعت فناوری و تناسب اندام
- نگهداری نام‌های برند، تکنولوژی‌ها و ابزارها به انگلیسی

**الزامات خروجی:**

- صفحات: تقریباً 40-50 صفحه با قالب‌بندی مناسب

- دسترسی‌پذیری: PDF برچسب‌دار با ساختار سرفصل مناسب

لطفاً یک سند صیقلی و حرفه‌ای ایجاد کنید که مناسب باشد برای:
- ارائه به سرمایه‌گذاران
