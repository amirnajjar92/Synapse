# Synapse Fit - Promotional Video Script for Descript

## Video Overview
**Duration:** 90-120 seconds  
**Format:** Professional app showcase with voiceover  
**Style:** Modern, dynamic, feature-focused  
**Tone:** Confident, innovative, empowering  
**Target Platform:** Social media, landing pages, app stores

---

## 📋 Script Structure

### Opening Hook (0-8 seconds)
**Visual:** Black screen → Synapse logo animation  
**Voiceover:**  
> "The future of fitness isn't just smart. It's intelligent."

**Music:** Upbeat, modern electronic (medium intensity)

---

### Problem Statement (8-18 seconds)
**Visual:** Quick cuts of common fitness frustrations:
- Confusing workout plans scattered across apps
- Generic advice that doesn't fit your body
- Lost progress data

**Voiceover:**

**On-screen text:** "AI-Powered Fitness OS"

## 🎬 Descript-Specific Production Notes

### Scene Organization
Create separate compositions for each feature segment:
1. `01_Opening_Hook`
2. `02_Problem_Statement`
3. `03_Solution_Intro`
4. `04_AI_Planner`
5. `05_Workout_Tracking`
6. `06_AI_Coach`
7. `07_Training_Studio`
8. `08_Ecosystem`
9. `09_Technology`
10. `10_Closing_CTA`

### Screen Recording Settings
- **Resolution:** 1080p (1920×1080) minimum
- **Frame Rate:** 60fps
- **Device Frame:** iPhone 14 Pro mockup or remove device frame for cleaner look
- **Orientation:** Portrait for app, landscape for overall composition
- **Recording Area:** Capture entire interface with no distractions

### Recording Checklist
Before recording each demo:
- [ ] Clear browser cache
- [ ] Disable browser extensions
- [ ] Use demo/test account with sample data
- [ ] Pre-populate forms to speed up demos
- [ ] Ensure animations are smooth (close other apps)
- [ ] Test audio levels
- [ ] Prepare cursor movements (smooth, deliberate)

### Voiceover Tips
- **Delivery:** Confident, energetic but not rushed
- **Pace:** 140-160 words per minute
- **Tone:** Professional yet approachable
- **Emphasis:** Stress power words (intelligent, personalized, real-time)
- **Pauses:** Strategic pauses before key features
- **Takes:** Record multiple takes per section

### B-Roll Suggestions
Supplement app demos with:
- Generic gym footage (no branding)
- Close-ups of people training
- Success stories (charts going up)
- Athletes using phones in gym
- Coach-client interactions
- Stock footage of technology/neural networks

### Text Overlays in Descript
Use Descript's text tool for:
- Feature callouts ("AI-Powered", "Real-Time Sync")
- Statistic highlights (if applicable)
- Social proof (if you have testimonials)
- Progress indicators (showing step 1 of 4)

### Transitions to Use
- **Between segments:** Smooth fade or slide
- **Feature highlights:** Quick zoom in + fade
- **Demo actions:** No transition (seamless recording)
- **Logo appearances:** Scale in with subtle glow

### Audio Layers
1. **Background Music:** 80% volume, ducking during voiceover
2. **Voiceover:** 100% volume, centered
3. **UI Sound Effects:** 40-60% volume (tap, swipe, notification sounds)
4. **Ambient Gym Sounds:** 20-30% volume (optional, adds realism)

### Export Settings
- **Format:** MP4 (H.264)
- **Resolution:** 1080p (1920×1080)
- **Bitrate:** High quality (8-12 Mbps)
- **Audio:** AAC, 192 kbps
- **Framerate:** 30fps or 60fps

### Social Media Variations
Create multiple cuts:
- **Full Version:** 90-105 seconds (YouTube, website)
- **60-Second Cut:** For Instagram, Facebook
- **30-Second Teaser:** For Twitter, TikTok
- **15-Second Clips:** Individual features for Stories

## 🎯 Quick Start Asset Preparation Guide

### Step 1: Capture Screenshots (30 minutes)
1. Open each demo page in browser
2. Use browser DevTools device mode (iPhone 14 Pro)
3. Take screenshots with Cmd+Shift+4 (Mac) or Snipping Tool (Windows)
4. Name files descriptively: `planner_ai_generating.png`

### Step 2: Record Screen Interactions (1 hour)
1. Use OBS Studio or QuickTime Player
2. Record each feature demo as separate clip
3. Keep recordings smooth and deliberate (no rushed clicking)
4. Record 2-3 takes of each interaction

### Step 3: Export Logo & Icons (15 minutes)
1. Open `/public/vectors/ai-icon.svg` in Illustrator/Figma
2. Export as PNG at multiple sizes (512px, 1024px, 2048px)
3. Ensure transparent backgrounds
4. Export app icons from `/public/icons/`

### Step 4: Gather Audio Assets (30 minutes)
1. Download background music from Epidemic Sound or YouTube Audio Library
2. Download UI sound effects from Freesound.org
3. Organize in `/05_Audio/` folder

### Step 5: Create Text Overlays (45 minutes)
1. Design feature callout cards in Canva or Figma
2. Use brand colors (black background, orange/blue accents)
3. Export as PNG with transparent backgrounds
4. Create 10-12 variations for different features

### Step 6: Upload to Descript (30 minutes)
1. Create new Descript project
2. Create folder structure as shown above
3. Upload all assets to respective folders
4. Verify all files imported correctly
